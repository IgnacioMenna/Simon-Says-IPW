# Simon-Says-IPW

A modern, web-based version of the classic **Simon Says** memory game.

---

## Description

Challenge your memory and reflexes by repeating the color sequence shown by the game.
Each round adds one more color, increasing the difficulty.

---

## Technologies Used

- **HTML5** – Structure and layout
- **CSS3** – Styling and responsive design
- **JavaScript (Vanilla)** – Core game logic and interactivity

---

## How to Play

1. Enter your name and click **Start Game**.
2. Watch carefully as the game highlights a sequence of colors.
3. Repeat the sequence by clicking the buttons in the same order.
4. Each successful level adds a new color to the sequence.
5. The game ends when you make a mistake.
